# Taryn Creator MVP

Premium creator command center for Taryn Truly.

## Features

- **Dashboard** — Morning greeting, follower stats, growth chart, activity feed, platform breakdown
- **Unified Inbox** — Multi-platform DMs, mental health mode toggle, thread view with reply
- **Analytics** — Engagement charts, daily reach, top performing content ranking

## Design System

- **Fonts:** Playfair Display (headings) + Inter (body)
- **Palette:** Cream `#FAF7F2` · Rose `#E8A598` · Clay `#C4856A` · Dark `#2C1810`
- **Cards:** White with soft shadow, 16px border radius

## Stack

- Next.js 15 (App Router)
- TypeScript
- Recharts (data visualization)
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
